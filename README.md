# Django Girls Amsterdam Blog

A Django-based blog application used as a teaching reference during my role as a **Coach / Mentor** at Django Girls Amsterdam.

---

## Role

I contributed as a **coach / mentor** at Django Girls Amsterdam, where I guided beginner developers in building their first Django application.

Key responsibilities included:

- Mentoring participants with little to no prior programming experience
- Explaining backend concepts with a focus on underlying principles and system thinking
- Assisting with debugging runtime, environment, and configuration issues
- Helping participants understand Django architecture and application structure
- Encouraging clean code practices and structured problem-solving approaches

This experience reflects my ability to communicate complex backend concepts clearly while supporting developers in a hands-on learning environment.

---

## Engineering Perspective

While the project is based on a beginner-friendly tutorial, my focus as a coach was to introduce real-world backend engineering thinking, including:

- Understanding Django’s request–response lifecycle
- Structuring applications for maintainability and clarity
- Identifying and debugging common issues in local development environments
- Explaining trade-offs in design decisions rather than only following implementation steps

This reflects my approach as a backend engineer: building systems with a strong understanding of how they work internally.

---

## Technical Scope

This project covers foundational backend and full-stack concepts:

- Client-server architecture and how the web works
- Python3 environment setup and Django framework initialization
- Django MVT architecture and request-response lifecycle
- ORM usage and database abstraction with SQLite
- URL routing and separation of concerns in views and templates
- CRUD operations with persistent data storage
- Basic frontend integration using HTML, CSS, and Bootstrap
- Deployment on PythonAnywhere

---

## Improvements Beyond Tutorial

- Improved code readability and structure for teaching clarity
- Provided debugging strategies for common beginner issues
- Emphasized best practices in project organization and backend thinking

## Tech Stack

Python, Django, SQLite, HTML, CSS, Bootstrap

## Quick Start

```bash
git clone https://github.com/sushmavankhede24/Django_Girls_Amsterdam_blog.git
cd Django_Girls_Amsterdam_blog
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```

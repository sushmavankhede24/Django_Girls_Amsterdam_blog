# Django Girls Amsterdam Blog

A Django-based blog application created as a teaching reference during my role as a **Coach / Mentor** at Django Girls Amsterdam.

---

## Role

I contributed as a **coach / mentor** at Django Girls Amsterdam, where I guided beginner developers in building their first Django application.

Key responsibilities included:

- Mentoring beginners with no prior programming experience
- Explaining backend concepts by focusing on underlying principles and system design thinking rather than only step-by-step instructions
- Assisting with debugging and resolving runtime and configuration issues
- Helping participants understand Django architecture and project structure
- Encouraging clean code practices and structured backend thinking

This project reflects my ability to mentor developers while applying and communicating core backend engineering concepts effectively.

---

## Technical Scope

This project covers foundational backend and full-stack concepts:

- Client-server architecture and fundamentals of how the web works
- Python 3 environment setup and Django framework initialization
- Django application design and structure
- MVC pattern implementation using models, views, URLs, and templates
- CRUD operations with database-backed persistence using SQLite
- Basic frontend integration using HTML and CSS
- UI enhancement using Bootstrap and Google Fonts
- Deployment on PythonAnywhere

---

## Tech Stack

Python, Django, SQLite, HTML, CSS, Bootstrap

---

## Quick Start

```bash
git clone https://github.com/sushmavankhede24/Django_Girls_Amsterdam_blog.git
cd Django_Girls_Amsterdam_blog
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```
---
## Access the application:

http://127.0.0.1:8000/
